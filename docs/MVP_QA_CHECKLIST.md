## 실제 데이터 fallback

### 25) 가격/이미지/태그 없는 데이터 fallback
- **체크 항목**: 결측 데이터가 UI를 깨지 않음
- **기대 동작**:
  - 가격 없음 → `가격 정보 없음` + `요금 정보 준비 중` 톤으로 표시
  - 이미지 없음 → placeholder
  - 홈페이지/예약/전화 없음 → 버튼 숨김
  - 태그 없음 → 태그 영역 숨김
  - 설명 없음 → `기본 정보가 준비 중입니다.`
- **현재 상태**: ⬜
- **수정 필요 여부**: ⬜
