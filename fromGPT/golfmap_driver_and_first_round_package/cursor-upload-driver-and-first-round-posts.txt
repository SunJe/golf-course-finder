Cursor 작업 지시문: GolfMap 블로그 글 3개 추가

목표:
GolfMap Korea 블로그에 아래 3개 글을 추가한다.

추가할 글
1. 첫 골프장, 머리올릴 때 필요한 준비물 체크리스트: 처음 필드 나가기 전 꼭 챙길 것
2. 남자 입문용 드라이버 추천: 핑 G425 MAX, 테일러메이드 SIM2 MAX, 야마하 인프레스 UD+2 비교
3. 여자 입문용 드라이버 추천: 젝시오 레이디스, 핑 G Le, 온오프 레이디 비교

중요 원칙
- Cursor는 판단하지 말고 아래 원고를 그대로 반영한다.
- 문장을 임의로 요약하거나 삭제하지 않는다.
- 제품 판매 링크를 넣지 않는다.
- 광고성 쇼핑 콘텐츠처럼 보이게 하지 않는다.
- 본문과 관련 글은 실제 존재하는 slug만 연결한다.
- 이미지에는 텍스트, 로고, 워터마크가 없으므로 블로그 대표/본문 이미지로 사용한다.
- 기존 블로그 스타일과 메타데이터 형식에 맞춰 적용하되, 핵심 원고 문구는 바꾸지 않는다.

작업 순서
1. 현재 workspace가 golf-course-finder인지 먼저 확인한다.
2. git status 로 기존 변경사항을 확인한다.
3. 위 이미지 6개를 public/promo-assets/blog/source/ 에 복사한다.
4. 현재 블로그 데이터 소스(lib/blogPosts.ts 또는 사용 중인 블로그 콘텐츠 파일)에 글 3개를 추가한다.
5. 각 글의 title, slug, excerpt, category, tags, coverImage/thumbnail/ogImage, 본문 내용을 원고에 맞춰 반영한다.
6. 이미지 삽입 위치는 각 원고의 “이미지 삽입 위치”를 그대로 따른다.
7. h1은 페이지 제목 하나만 사용하고, 주요 소제목은 h2, 모델별 소제목은 h3로 구조화한다.
8. 표가 있는 글은 마크다운 테이블 또는 기존 컴포넌트에 맞는 비교 표 UI로 추가한다.
9. “빠른 결론”은 상단 요약 박스 형태로 잘 보이게 넣되, 기존 사이트 톤에 맞춘다.
10. 관련 글 섹션은 실제 존재하는 slug만 연결한다.
